# Poo - Клиент для установки и управления SparkPanel

Poo - это клиентское приложение для установки и управления веб-панелью SparkPanel на серверах с ОС Linux. Приложение предоставляет интуитивный графический интерфейс для автоматической установки и настройки панели управления серверами.

## Основные возможности

### 1. Установка SparkPanel
- Автоматическая установка LEMP стека (Linux, Nginx, MySQL, PHP)
- Разворачивание веб-панели управления
- Настройка веб-сервера и безопасности
- Поддержка аутентификации по паролю и SSH ключу

### 2. Управление панелью
- Встроенный браузер для доступа к панели
- Управление серверами Minecraft
- Мониторинг ресурсов сервера
- Управление плагинами и модификациями
- Резервное копирование и восстановление
- Планировщик задач
- Система уведомлений
- Расширенные функции безопасности

### 3. Мониторинг ресурсов
- **Графики использования ресурсов:**
  - ЦПУ (CPU)
  - Оперативная память (RAM)
  - Дисковое пространство
- **Мониторинг сетевой активности:**
  - Трафик в реальном времени
  - Количество отправленных/полученных пакетов
- **Метрики производительности Minecraft-сервера:**
  - Количество онлайн-игроков
  - TPS (Ticks Per Second)
  - Загруженные чанки
  - Сущности в мире
  - Время работы сервера
- **Система оповещений:**
  - Предупреждения при достижении пороговых значений
  - Критические уведомления о проблемах
  - Настройка порогов для различных метрик

### 4. Безопасность
- Двухфакторная аутентификация
- Ограничения доступа по IP-адресам
- Аудит действий пользователей
- Роли и разрешения пользователей

### 5. Уведомления
- Поддержка Email уведомлений
- Интеграция с Telegram
- Интеграция с Discord
- Настройка типов уведомлений

### 6. Улучшение производительности
- **Оптимизация базы данных:**
  - Индексы для ускорения запросов
  - Оптимизированные таблицы
- **Кэширование данных:**
  - Кэширование часто запрашиваемых данных
  - Автоматическая очистка кэша
  - Мониторинг использования кэша
- **Фоновая обработка задач:**
  - Очередь задач для тяжелых операций
  - Приоритезация задач
  - Мониторинг состояния задач
- **Система логирования и аналитики:**
  - Детальное логирование действий
  - Аналитика по пользователям и операциям
  - Очистка старых логов

### 7. Дополнительные возможности
- **Система документации и справки:**
  - Подробная документация по всем функциям
  - Часто задаваемые вопросы
  - Поиск по документации
  - Руководство пользователя в формате PDF
- **API для интеграции с другими сервисами:**
  - RESTful API для управления серверами
  - Аутентификация по API ключу
  - Поддержка основных операций (запуск, остановка, управление плагинами и т.д.)
- **Система обновлений панели:**
  - Автоматическая проверка обновлений
  - Скачивание и установка обновлений
  - История обновлений
- **Многопользовательская поддержка с разграничением прав:**
  - Роли пользователей (администратор, модератор, пользователь)
  - Гибкая система разрешений
  - Управление пользователями

## Технические характеристики

### Клиентская часть (Poo)
- Язык программирования: Python 3.8+
- Графический интерфейс: PyQt5
- Библиотеки: Paramiko (SSH), psutil (мониторинг)
- Поддерживаемые ОС: Windows 7+, Linux

### Серверная часть (SparkPanel)
- Веб-сервер: Nginx
- Язык программирования: PHP 8.2
- База данных: MySQL
- Поддерживаемые ОС: Ubuntu 20.04+

## Установка

### Требования к клиенту
- Python 3.8 или выше
- pip (менеджер пакетов Python)

### Установка зависимостей клиента
```bash
pip install -r client_poo/requirements.txt
```

### Запуск клиента
```bash
cd client_poo
python main.py
```

## Использование

1. Запустите клиентское приложение
2. Перейдите на вкладку "Установка SparkPanel"
3. Введите параметры подключения к вашему VDS серверу
4. Нажмите "Установить SparkPanel"
5. После установки перейдите на вкладку "Управление панелью"
6. Откройте встроенный браузер для доступа к панели
