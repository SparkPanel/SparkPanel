# Быстрый запуск SparkPanel v1.3 для тестирования

## Шаг 1: Установка зависимостей

Откройте терминал в папке проекта и выполните:

```bash
npm install
```

## Шаг 2: Запуск в режиме разработки

Для тестирования в браузере используйте команду:

```bash
npm run dev
```

Панель будет доступна по адресу: **http://localhost:5000**

## Шаг 3: Вход в панель

**Учетные данные по умолчанию:**
- **Username**: `adplayer`
- **Password**: `0000`

## Важно для тестирования

### Если Docker НЕ установлен:
- ❌ Нельзя будет создавать/запускать серверы
- ❌ Нельзя просматривать логи в реальном времени
- ✅ Можно просматривать UI и навигацию
- ✅ Можно управлять настройками пользователя
- ✅ Можно просматривать структуру панели

### Если Docker установлен и запущен:
- ✅ Все функции работают полностью
- ✅ Можно создавать и управлять серверами
- ✅ Можно просматривать логи в реальном времени
- ✅ Можно управлять файлами

## Проблемы при запуске?

### Порт 5000 занят?
Измените порт:
```bash
PORT=8080 npm run dev
```
Тогда панель будет доступна по адресу: **http://localhost:8080**

### Ошибки при установке?
Убедитесь, что у вас установлены:
- Node.js версии 18 или выше: `node --version`
- NPM версии 8 или выше: `npm --version`

## Что можно протестировать без Docker:

## Что нужно для полного функционала:

## Остановка сервера

Нажмите `Ctrl+C` в терминале для остановки сервера разработки.
